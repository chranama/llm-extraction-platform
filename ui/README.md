# UI

## Purpose
Frontend interface for interacting with generate/extract/admin functionality.

## Key Entrypoints
- `ui/src/`
- `ui/e2e/`

## Run/Test
```bash
cd ui
npm test
npm run test:coverage
```

## Dependencies
- Calls backend API configured via `config/ui.json` and build-time env vars.
